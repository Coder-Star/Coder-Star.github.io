### 1、快速排序

```java
class Solution {
    public int[] sortArray(int[] nums) {
        if (nums == null || nums.length < 2) {
          return nums;
        }
        quickSort(nums,0,nums.length-1);
        return nums;
    }
    // 快速排序
    public void quickSort(int[] nums,int begin,int end){
        if(begin >= end){
            return;
        }
        int n = partation(nums,begin,end); 
        quickSort(nums,begin,n);
        quickSort(nums,n+1,end);
    }
    // 获取基准数位置
    public int partation(int[] nums,int begin,int end) {
        // 取第一个数
        int privot = nums[begin];
        while(begin < end){
            // 从右向左找小于privot的数来填begin
            while(nums[end] >= privot && begin < end){
                end--;
            } 
            nums[begin] = nums[end];
             // 从左向右找大于或等于privot的数来填end
            while(nums[begin] < privot && begin < end){
                begin ++;
            }
            nums[end] = nums[begin];
        }
        nums[end] = privot;
        return end; 
    }
}
```

## 二叉树

### 深度、宽度

**深度**

```java
// 使用队列，层次遍历二叉树。在上一层遍历完成后，下一层的所有节点已经放到队列中，此时队列中的元素个数就是下一层的宽度。以此类推，依次遍历下一层即可求出二叉树的最大宽度
  public static int getMaxDepth(TreeNode root) {
        if (root == null)
            return 0;
        else {
            int left = getMaxDepth(root.left);
            int right = getMaxDepth(root.right);
            return 1 + Math.max(left, right);
        }
    }
```

**宽度**

```java
 public static int getMaxWidth(TreeNode root) {
        if (root == null)
            return 0;

        Queue<TreeNode> queue = new ArrayDeque<TreeNode>();
        int maxWitdth = 1; // 最大宽度
        queue.add(root); // 入队

        while (true) {
            int len = queue.size(); // 当前层的节点个数
            if (len == 0)
                break;
            while (len > 0) {// 如果当前层，还有节点
                TreeNode t = queue.poll();
                len--;
                if (t.left != null)
                    queue.add(t.left); // 下一层节点入队
                if (t.right != null)
                    queue.add(t.right);// 下一层节点入队
            }
            maxWitdth = Math.max(maxWitdth, queue.size());
        }
        return maxWitdth;
    }
```

### 台阶问题

1、每次可以跳一级或二级台阶，n 级台阶有多少种方式?  
2、如果每次可以跳 1->n 次台阶，有多少种方式

针对第一个问题，倒着分析，第 n 级台阶的可能性是第 n-1 级的可能性 加上 n-2 级的可能性，所以就有 f(n) = f(n-1) + f(n-2),这是一个很明显的递归逻辑；
第二个问题：  
f(n) = f(n-1) + f(n-2) + ... f(1)  
f(n-1) = f(n-2) + f(n-3) + f(1)  
两者合并
f(n) = 2f(n-1) = 2<sup>n-1</sup>f(1)
