## 1、设置 Tomcat 部分参数

### 1、可在/bin/catalina.bat 文件中设置以下参数

```vim
//set local
set JAVA_OPTS=-Xms2048m -Xmx2048m //设置tomcat启动内存大小
set JAVA_HOME=D:\Java\jdk1.8.0_60  //设置tomcat所用jdk
set title=OA8932 //设置tomcat窗口title名称
```
